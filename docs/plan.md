# 企画書 (Project Plan) - アセットアロケーション管理ツール "Elena"

## 1. プロジェクト概要
**プロジェクト名**: Elena (エレナ)
**目的**: 個人投資家がアセットアロケーション（資産配分）を最適化・維持するために、主要な資産クラスの市場動向を一目で把握できるダッシュボードを提供する。

## 2. 背景・課題
*   **課題**: 複数の証券会社やニュースサイトを巡回しないと、株式・債券・コモディティ等の全体感が掴みにくい。
*   **解決策**: 1画面で「世界の株式」「債券金利」「不動産」「金・原油」の状況を俯瞰できるツールを作成する。

## 3. 提供価値
*   **「一覧性」**: クラスごとの相関関係（株安・債券高など）を瞬時に視認できる。
*   **「高速性」**: エッジコンピューティング (Cloudflare Workers + WASM) による爆速な表示。
*   **「シンプル」**: 余計な情報を削ぎ落とし、資産配分の意思決定に必要な指標のみを表示。

## 4. ターゲットユーザー
*   分散投資を行っている、またはこれから行いたい個人投資家。
*   市場全体の体温感を毎日手軽にチェックしたいユーザー。
